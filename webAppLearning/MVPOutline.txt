- Web app created using flask from which users can interact with the program for the whole of the workflow

		- Preview of uploaded spreadsheet
	- Options (% of data used for training vs test, cycles ran, etc.)
- Can upload excel spreadsheet with data

	- Output equation of line of best fit
	- Make prediction
- Plot data before and after
	- Plot prediction
